The littleURL is an URL shortener. It receives a web address and returns an slug (the part of the littleURL's URL after the slash symbol); this is depicted in the figure 1.
It can "unshorten" an URL. And can track how many clicks the URL had received.

figure 1 - How the shortening works? 
example_of_an_extra_large_url.com → process → ctn.rf.gd/slug
